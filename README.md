# Portfolio Website

A personal portfolio website built to showcase my skills, projects, and achievements.

**Live:** [lk-gambhir.github.io/portfolio](https://lk-gambhir.github.io/portfolio/)

## Sections

- **Home** — Intro with typing text animation
- **About** — Brief bio with social links
- **Qualification** — Education history
- **Skills** — Technical skills and experience
- **Projects** — Donation Website, Portfolio Website
- **Achievements** — LeetCode, Hackathons, JEE Mains
- **Contact** — Phone, email, LinkedIn, GitHub

## Tech Stack

- HTML, CSS, JavaScript
- jQuery
- Typed.js (typing animation)
- Owl Carousel (achievements slider)
- Font Awesome (icons)

## Run Locally

Open `index.html` directly in a browser, or start a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`
